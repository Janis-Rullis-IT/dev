# Nanobox

* https://docs.nanobox.io/cli/

## Install

* https://dashboard.nanobox.io/download

```shell
nanobox config
nanobox login
nanobox remote ls
nanobox remote add my-dev
nanobox remote add my-prod production
nanobox remote ls
```

App Name     | Alias
------------------------
my-dev   | default
my-prod  | production
